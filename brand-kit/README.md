# Harper Brand Kit

`harper-brand-kit.zip` contains Harper's brand guidelines as 46 PDF files: `Harper_Brand Guidelines.pdf` plus the numbered pages `Harper_Brand Guidelines-1.pdf` through `Harper_Brand Guidelines-45.pdf`. Together they cover Harper's logo usage, color palette, typography, and layout standards.

Use these guidelines alongside the subjectivity rendering skill whenever you produce customer-facing material: the skill governs what copy is allowed to say, and the brand kit governs how it looks. Unzip the archive locally to browse individual pages.
